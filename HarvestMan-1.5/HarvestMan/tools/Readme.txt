These are some utilities for working with HarvestMan
and generated files.

1. genconfig.py - 	An interactive command-line config file 
                        generation script .
2. projectreader.py - 	Inspect harvestman project files.
3. sitebuilder.py-      Generate a website directory from the cache file
                        of its project.
4. proxyobfuscator.py-  Interactive tool to create proxy settings
                        to be used in the config file.
5. headerdump.py-       Dump URL headers in a DBM file from
                        the project cache.
6. convertconfig.py-    Convert configuration files between text
                        and xml formats.
